# 代码自动生成器

一直以来都想想一个代码生成的项目，最近闲下来了，完善一下,这是一个基于gradle 的 springBoot项目，基于 web 的

## 技术栈：

+ Gradle
+ SpringBoot
+ Bootstrap

## 完成功能

- [x] 一键复制
- [x] 页面完成
- [x] Linux 环境部署

```
└── src
    ├── main
    │   ├── java
    │   │   └── cn
    │   │       └── okjava
    │   │           └── bennycodegenerator
    │   │               └── generator
    │   │                   ├── bean
    │   │                   ├── config
    │   │                   ├── repository
    │   │                   ├── rest
    │   │                   └── service
    │   │                       └── impl
    │   └── resources
    │       ├── static
    │       │   └── css
    │       └── templates
    │           └── tmpl
    └── test
        ├── java
        │   └── cn
        │       └── okjava
        │           └── bennycodegenerator
        │               └── generator
        │                   ├── config
        │                   ├── repository
        │                   ├── rest
        │                   └── service
        │                       └── impl
        └── resources
```
